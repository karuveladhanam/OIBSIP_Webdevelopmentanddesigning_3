 🌡️ Temperature Converter
A simple and responsive temperature converter web application built using HTML, CSS, and JavaScript. This project allows users to easily convert temperature values between Celsius, Fahrenheit, and Kelvin with a clean and user-friendly interface.

✨ Features
🌡️ Temperature Conversion – Convert between Celsius, Fahrenheit, and Kelvin
🧮 Input Validation – Handles invalid or empty inputs
⚡ Instant Calculation – Quick and accurate results
📱 Responsive Design – Works on mobile and desktop
🎨 Clean UI – Minimal and modern design

🛠️ Technologies Used
HTML5
CSS3
JavaScript

🚀 How to Run the Project
Clone the repository:
Open the folder
Run the project:
Double-click **index.html**
OR
Open with any browser

🎯 Functionality
Enter temperature value → Input field accepts numeric values
Select unit → Choose Fahrenheit, Celsius, or Kelvin
Click **Convert** → Displays converted result
Invalid input → Shows error message
